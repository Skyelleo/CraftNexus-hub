# CraftNexus Hub

CraftNexus Hub is a modern desktop application designed to simplify Minecraft server management.

It provides an all-in-one interface for managing servers, mods, configurations, backups, and performance — without the usual complexity.

---

## ✨ Features

* 🖥️ Multi-server management
* ⚡ Start / Stop / Restart controls
* 📜 Live console with log filtering
* ⚙️ Config file editor (visual + raw)
* 📦 Mod management with compatibility checks
* 💾 Backup system with restore preview
* 👥 Player management (whitelist, ops, bans)
* 📊 Resource monitoring (CPU & RAM)

---

## 🎯 Goal

CraftNexus Hub aims to make running Minecraft servers easier, faster, and more accessible — whether you're hosting for friends or managing larger setups.

---

## 💰 Business Model

The application follows a freemium model:

* Core features are free
* Advanced features are available in a one-time Pro upgrade

CraftNexus Hub does not sell or redistribute mods. It respects mod authors by linking to official sources.
